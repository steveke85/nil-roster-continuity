# Data

The dataset used in this project was sourced from KenPom.com, a subscription-based 
college basketball analytics service. Due to licensing restrictions, the raw data 
is not included in this repository.

To reproduce this analysis, you will need a KenPom subscription. Export the 
following fields for all Division I teams, seasons 2008–2026 (excluding 2020–2021):

- Season
- Team Name
- AdjEM (Adjusted Efficiency Margin)
- Continuity
